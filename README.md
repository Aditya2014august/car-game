# 🏎️ High-Graphics 3D Car Game

An immersive, hardware-accelerated 3D car driving simulator built entirely with **HTML5** and **Three.js**. This project features smooth driving physics, a responsive steering wheel UI, and dynamic lighting.

## 🎮 Controls
* **W / Up Arrow:** Accelerate
* **S / Down Arrow:** Brake / Reverse
* **A / Left Arrow:** Steer Left
* **D / Right Arrow:** Steer Right

## ✨ Features
* **High Performance:** Uses WebGL for smooth 60FPS gameplay.
* **Interactive Dashboard:** Includes a speed-o-meter and a steering wheel that rotates as you turn.
* **Atmospheric Visuals:** Realistic fog, shadows, and reflective materials.
* **Responsive Design:** Automatically adjusts to fit any screen size.

## 🛠️ Built With
* [Three.js](https://threejs.org/) - 3D Engine
* HTML5 & CSS3
* JavaScript (ES6+)
